# PRODIGY_CS_03

## Software Development Internship - Task 3

### Task

Simple Contact Management System

### Description

This is a Python-based Contact Management System developed as part of the Prodigy InfoTech Software Development Internship. The program allows users to add, view, edit, and delete contacts. Contact details such as name, phone number, and email address are stored in a JSON file for persistent storage.

### Features

- Add new contacts
- View saved contacts
- Edit existing contacts
- Delete contacts
- Store contact data in a JSON file

### Technologies Used

- Python
- JSON
